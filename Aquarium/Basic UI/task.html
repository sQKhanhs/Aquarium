Description

The animal world is very diverse. A local aquarium has hired you to familiarize people with their resident animals.

Your job is to create a page for those animals displaying an image, the species name, and a description.

In this first stage, let's create the layout for one page so that your employers can validate the page look before making the project more complex with the pagination scheme.

Objectives

  1. Receive input from tests

    • Tests may pass the data needed for this stage through Intent.extras. To make manual testing possible, there will also be default values for the data, which will be used if no input is given;

    • Use the imageAnimals key to retrieve a serializable from intent.extras. Cast that serializable to List<Int>. If there is no imageAnimals input, use the following as the default value:

      listOf(R.drawable.image1)

      The imageAnimals is a list of resource images for the animals in the aquarium;

    • Use the nameAnimals key to retrieve a serializable from intent.extras. Cast that Serializable to List<String>. If there is no nameAnimals input, use the following as the default value:

      listOf("Koi Carp")

      The nameAnimals is a list of names for the animals in the aquarium;

    • Use the descriptionAnimals key to retrieve a serializable from intent.extras. Cast that serializable to List<String>. If there is no descriptionAnimals input, use the following as the default value:

      listOf(
        "These colorful, ornamental fish are a variety of the Amur carp. " +
        "They were originally found in Central Europe and Asia, " +
        "but they’ve spread to many other parts of the world. " +
        "Koi carp are popular with breeders, and there are currently over 100 varieties " +
        "created through breeding.\n" +
        "\n" +
        "The average age of a koi carp can vary based on the part of the world it’s bred in. " +
        "Carps bred outside of Japan have an average lifespan of around 15 years," +
        " while carps bred in Japan can live 40 years or more. The oldest koi carp on record," +
                " which was a fish named Hanako, reportedly lived for 226 years!",
)

      The descriptionAnimals is a list of descriptions for the animals in the aquarium;

    • Data on a particular animal is distributed in those three lists. Each animal has its index. So at index 0, you will get the data (image, name, description) about the same animal. In this stage, use the first element of the list, but in future stages, we will make a pagination scheme and have a different animal for each page.

  2. Add CardView with the card_view id in activity_main.xml

    • set the elevation to 10

    • set the cornerRadius to 15

    • this card_view will create a "page" for the animal. You can use a layout inside card_view to arrange multiple views inside.

    • you can use this documentation link for reference

  3. Place ImageView with the image_view id inside card_view with scaleType centerCrop. This view will display the imageAnimals data

  4. Place TextView with the tv_name id inside card_view with fontFamily monoscape, textStyle bold, and textAppearance TextAppearance.MaterialComponents.Body1. This view will be used to display nameAnimals data

  5. Place NestedScrollView with the scroll_view id inside card_view. This view will ensure the description text is readable even in the case of a very, very long description.

  6. Place TextView with the tv_description id inside scroll_view with fontFamily sans-serif and textAppearance TextAppearance.MaterialComponents.Body2. This view will display the descriptionAnimals data.
