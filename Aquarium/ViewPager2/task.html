Description

In this stage, you will learn how to work with ViewPager2 and the RecyclerView adapter to configure it.

You are hanging out with colleagues and notice a child playing with the app. The kid can't read and goes straight to swiping. In the kid's magical world, it is how it goes; swipes move the world around them.

You return home and decide to make a pagination scheme with swiping instead. While researching the subject, you discover an Android library named ViewPager2 will give you what you want with more or less the same kind of code that you would need for RecyclerView.

You think you will probably have to adapt slightly to the layout structure. Luckily all the page content was already inside the CardView, and a simple cut-and-paste from that into a new layout page_item.xml and ViewPager2 would make that change come true.

Objectives
  1. Remove the buttons!!!;
  2. Create an item of ViewPager2. You can copy the page with CardView;
  3. Place ViewPager2 in activity_main.xml; give it the viewpager2 id;
  4. Create an adapter for ViewPager2. Use the RecyclerView Adapter.
  5. Configure the adapter to ViewPager2.

NOTE: There will be no change in receiving data.
